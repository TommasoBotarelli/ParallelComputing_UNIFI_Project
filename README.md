# Parallel Computing Course Project 

Parallel Computing course project held by Professor Marco Bertini (UNIFI). 
There are three different folders one for each topic containing the code and the results of the experimentation.
In the *reports* folder you can find the reports of each experiment.

Exam projects:
- *Time Series Pattern Recognition* (C++ & OpenMP) [link](https://github.com/TommasoBotarelli/ParallelComputing_UNIFI_Project/blob/master/reports/pattern_recognition.pdf)
- *Kernel Image Processing* (C++ & CUDA) [link](https://github.com/TommasoBotarelli/ParallelComputing_UNIFI_Project/blob/master/reports/kernel_image_processing.pdf)

Laboratory: *Image Augmentation* (Python Multiprocessing) [link](https://github.com/TommasoBotarelli/ParallelComputing_UNIFI_Project/blob/master/reports/image_augmentation.pdf)
